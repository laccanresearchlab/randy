FAPEAL - Pronem (2012-2015)
Para a implementação de soluções de monitoramento e controle de um edifício inteligente, podemos utilizar diferentes abordagens, métodos e modelos. No entanto, tais abordagens consideram soluções e validações específicas para cada problema. Devido a essa especificidade, a comparação entre diferentes soluções, técnicas empregadas, a proposição e validação de novas técnicas torna-se uma tarefa difícil. Nessa direção, esse projeto vislumbra o desenvolvimento de um testbed, chamado de Smart-CPMAT, para implantação de diferentes dispositivos presentes num edifício inteligente combinado com a implementação de um arcabouço que possa padronizar e ser utilizado como base para: projeto, desenvolvimento, teste e validação de soluções para o monitoramento de edifícios inteligentes.

A API SensorNet permite o acesso de leitura programático dos dados dos sensores instalados no prédio do CPMAT. No momento, autenticação por OAuth não é requerida, mas será num futuro próximo.
Se a tua intenção é monitorar, ou fazer uso dos nossos dados em tempo real, essa com certeza é a melhor opção para você.
